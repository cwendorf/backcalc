# [`backcalc`](https://github.com/cwendorf/backcalc)

## Reference

This section is the reference for `backcalc`. Each page documents one exported function, including usage, arguments, and return values.

### Backcalculate Methods

- [backcalc_anova](./backcalc_anova.md): Handles between-subjects, within-subjects, factorial, and mixed designs, with optional sphericity corrections.
- [backcalc_coeff](./backcalc_coeffs.md): Handles unstandardized and standardized coefficients.
- [backcalc_corrs](./backcalc_corrs.md): Handles correlation coefficients using Fisher's z-transformation. It supports both one-sample and two-sample correlation comparisons.
- [backcalc_means](./backcalc_means.md): Handles one-sample, paired, and two-sample cases.
- [backcalc_medians](./backcalc_medians.md): Handles medians ad robust measures of spread such as interquartile range (IQR), median absolute deviation (MAD), and range.
- [backcalc_props](./backcalc_props.md): Handles proportions for one-sample, paired, and two-sample cases.
- [backcalc_ratios](./backcalc_ratios.md): Handles ratio-type measures such as odds ratios or risk ratios.
- [backcalc_standardized_means](./backcalc_standardized_means.md): Handles standardized mean differences and related effect size statistics for one-sample, paired, and two-sample designs.
