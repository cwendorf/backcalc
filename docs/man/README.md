# [`backcalc`](https://github.com/cwendorf/backcalc/)

## Reference

This section is the reference for `backcalc`. Each page documents one exported function, including usage, arguments, and return values.

### Means and Analysis of Variance

- [backcalc_means](./backcalc_means.md): Backcalculates means and mean differences.
- [backcalc_standard](./backcalc_standard.md): Backcalculates standardized mean differences and related effect sizes.
- [backcalc_anova](./backcalc_anova.md): Backcalculates ANOVA results for between, within, factorial, and mixed designs.

### Nonparametric Statistics

- [backcalc_medians](./backcalc_medians.md): Backcalculates medians and median differences using robust spread inputs.
- [backcalc_props](./backcalc_props.md): Backcalculates proportions and proportion differences.
- [backcalc_ratios](./backcalc_ratios.md): Backcalculates ratio-type effects such as odds ratios and risk ratios.

### Correlation and Regression

- [backcalc_corrs](./backcalc_corrs.md): Backcalculates correlations and correlation differences.
- [backcalc_coeffs](./backcalc_coeffs.md): Backcalculates coefficients in bivariate regression models.
- [backcalc_multreg](./backcalc_multreg.md): Backcalculates coefficients in multiple regression models.
