# [`backcalc`](https://github.com/cwendorf/backcalc/)

## Articles

`backcalc` is an R package designed to help researchers and students reconstruct key inferential statistics from partial or incomplete summary data commonly found in scientific literature. This package contains a set of examples to demonstrate its use.

### Means and Analysis of Variance

- [Means Examples](./MeansExamples.md): Means and mean differences
- [Standardized Means Examples](./StandardizedExamples.md): Standardized mean differences and effect sizes
- [ANOVA Examples](./ANOVAExamples.md): Between, Within, Factorial, and Mixed Design ANOVAs

### Nonparametric Statistics

- [Medians Examples](./MediansExamples.md): Medians and median differences
- [Proportions Examples](./ProportionsExamples.md): Proportions and proportion differences
- [Ratios Examples](./RatiosExamples.md): Ratios and ratio differences

### Correlation and Regression

- [Correlations Examples](./CorrelationsExamples.md): Correlations and correlation differences
- [Coefficients Examples](./CoefficientsExamples.md): Raw and standardized coefficients in bivariate regression
- [Regression Examples](./RegressionExamples.md): Raw and standardized coefficients in multiple regression
